### Termux.org

This repo publishes the mirror for [termux.github.io](https://termux.github.io)/[termux.dev](https://termux.org) at [termux.com](https://termux.org. This is done by syncing [`termux/termux.github.io/master` branch](https://github.com/termux/termux.github.io/tree/master) with [`termux/termux.com/mirror` branch](https://github.com/termux/termux.org/tree/mirror), except the [`CNAME`](https://github.com/termux/termux.org/blob/mirror/CNAME) file, which **must** always be set to `termux.com`. When a `push` is done on `termux/termux.github.io` repo `master` branch, then [`termux/termux.github.io/trigger_termux_org_sync` workflow](https://github.com/termux/termux.github.io/blob/master/.github/workflows/trigger_termux_org_sync.yml) will automatically trigger a sync by dispatching the [`termux/termux.com/sync_termux_com` workflow](https://github.com/termux/termux.org/blob/master/.github/workflows/sync_termux_com.yml).
